# Setup

### Environment Variables

```js
PORT=
HOSTNAME=
PASSWORD=
PG_USER=
```

### 1 - Instalando dependências

```bash
npm install
```

### 2 - Criando base de dados

```bash
npm run db:create
```

### 3 - Sincronizando migrações

```bash
npm run db:migrate
```

### 4 - Populando tabelas (Mocking)

```bash
npm run db:populate
```
